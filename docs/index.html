translation.io

A simple Web API to translate strings.

Collection

  • POST /collections
  • GET /collections/{CollectionId}
  • PUT /collections/{CollectionId}
  • DELETE /collections/{CollectionId}

  • Errors

POST /collections

Request

curl http://radiant-wave-4656.herokuapp.com/collections \
-d "name=My Collection"

Response

{
    "Collection": {
        "Id": "514148ed4ee25e0002000001",
        "Name": "My Collection",
        "Strings": null
    }
}
